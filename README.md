# ExpeditionAutoWebsite

Landing page for the [Expedition Auto](https://github.com/ManilShangle/ExpeditionAuto) Chrome extension.

Live at: **[manilshangle.github.io/ExpeditionAutoWebsite](https://manilshangle.github.io/ExpeditionAutoWebsite/)**

## Pages

| File | URL | Purpose |
|---|---|---|
| `index.html` | `/` | Main landing page |
| `privacy.html` | `/privacy.html` | Privacy policy (required for Chrome Web Store) |
| `contact.html` | `/contact.html` | Support FAQ + contact form |

## Contact Form

Submissions are handled by [Formspree](https://formspree.io). The form ID is set in `contact.html`.

## Deployment

Deployed automatically via GitHub Actions on every push to `main`. Workflow lives at `.github/workflows/deploy.yml`.
